03 リンクの設定

・a(anchor)タグを使い。href(hypertext reference)属性を追加。新規タブで開くにはtarget="_blank"と記入。

・ページ内リンクは、リンク先にid属性を設定後に、href属性に#id名。

・サイト内のページ外リンクは、リンク先のid属性の前にhtml等をつける、<a href="part10.php#id名">id先</a>


04 文字参照

・<h2>を表示するには、文字参照を使用する。小なり記号(<)は &(アンパサンド)+lt(less than)+;（セミコロン）で表し、大なり記号(>)は &+gt(greater than)+;で表す。

・コードを表現する際は code タグを使用する。

0502 チェックボックス

商品名

・input type = "checkbox"で作成。

・複数選択が可能。

・fieldsetで枠で囲める。legendで説明も付けられる。

・checkedで初期値複数選択可能。

0503 ラジオボタン
満足度

・input type = "radio" で作成。

・全てのボタンにname=""で同じ値を付けることで複数選択できなくなる。

・checkedで初期値で1つ選択可能。

0504 ドロップダウンリスト

・optionタブをselectタブで囲って作成。

・一番上を説明文にできる。

・selectタブにsize=3を加えて3行表示初期設定等が可能。ドロップダウンリストでなくなるので、リストボックスになる。

・デフォルトは1つだけ選択可能。selectタブにmultipleを加えると複数選択可能になるが、選択方法がctrl+等になるので、チェックボックスのほうがわかり易い。

・optionタブにselectedで初期設定可能。ただし、変更後、ブラウザで更新しても変更されなかった。一度タブを閉じればOK。
