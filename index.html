○リンクの設定

・a(anchor)タグを使い。href(hypertext reference)属性を追加。新規タブで開くにはtarget="_blank"と記入。

・ページ内リンクは、リンク先にid属性を設定後に、href属性に＃id名。

○文字参照

・<h2>を表示するには、文字参照を使用する。小なり記号(<)は &(アンパサンド)+lt(less than)+;（セミコロン）で表し、大なり記号(>)は &+gt(greater than)+;で表す。

・コードを表現する際は code タグを使用する。

○入力部品

input

・type属性text。単一行テキストボックス。placeholder使用。

・placeholderは使用しないほうがよい。他の方法ほど有益でなく、コンテンツに予期しない技術的な問題を引き起こす可能性がありとのこと。

・type属性text。初期値をvalue属性で記入可能。

・type属性password。初期値をvalue属性で記入可能。

・type属性number。初期値をvalue属性で記入可能。

・type属性date。初期値をvalue属性で記入可能。形式はvalue="2022-08-18"。

textarea

・改行を含む複数行のテキストボックス。

・labelとテキストボックスを結び付けるには、

  1. labelの中にテキストボックスを配置。
  2. for="id名"で関連付ける。

２つの方法がある。

・placeholderで例が記入可能。ただし、規定値をセットしておくと表示が見えなくなる。

・テキストエリアに初期値を記載できるが、タブも文字に含まれてしまうので注意。

チェックボックス

商品名

・input type = "checkbox"で作成。

・複数選択が可能。

・fieldsetで枠で囲める。legendで説明も付けられる。

・checkedで初期値複数選択可能。

ラジオボタン

満足度

・input type = "radio" で作成。

・全てのボタンにname=""で同じ値を付けることで複数選択できなくなる。

・checkedで初期値で1つ選択可能。

ドロップダウンリスト

・optionタブをselectタブで囲って作成。

・一番上を説明文にできる。

・selectタブにsize=3を加えて3行表示初期設定等が可能。ドロップダウンリストでなくなるので、リストボックスになる。

・デフォルトは1つだけ選択可能。selectタブにmultipleを加えると複数選択可能になるが、選択方法がctrl+等になるので、チェックボックスのほうがわかり易い。

・optionタブにselectedで初期設定可能。ただし、変更後、ブラウザで更新しても変更されなかった。一度タブを閉じればOK。

ボタン

  1. <button>ボタン<button>
  2. <input type="submit" value="ボタン">

とする方法がある。

・<button>を使用すれば、画像も使用可能。

・disabledで無効化できる。
