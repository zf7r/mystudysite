  24. @counter-style を使わずに、疑似要素を使って括弧数字にする方法。
    1. li 要素に背景色をつけて、違いがわかりやすいようにした。
    2. こちらは位置調整しない場合。
    3. 2行になると字下げがされていない。
  25. 疑似要素を使って括弧数字にした場合の位置の調整。 
    【css 調整なし】
          .p00501ol {  ←クラス名は適当 
          list-style: none;    ← リストスタイル不使用
        }
        .p00501ol li {
          width:500px;  ←2行にするため幅を狭くした
          counter-increment: my-counter;  ←"my-counter" を 1 増加
          background-color: aquamarine;  ←確認しやすいように背景色追加
        }
        .p00501ol li::before {
          content: "【" counter(my-counter) "】 ";
        }
    1. margine-left: -60px;として疑似要素をli要素の外に出した。
    2. display: inline-block;にして、疑似要素とli要素を横並びにする。
    3. 疑似要素の幅を調整した。1桁だと間が大きいが、2桁でもOK。
    【css 調整あり】
        .p00501ol2 {
    margin-left: -20px;  ←li要素を左に移動
    list-style: none;
  }
  .p00501ol2 li {
    width:500px;
    counter-increment: my-counter;
    background-color: aquamarine;
  }
  .p00501ol2 li::before {
    content: "【" counter(my-counter) "】 ";
    width: 60px;  ←疑似要素の幅を調整
    margin-left: -80px;  ←li要素の左に疑似要素を移動
    display: inline-block;  ←疑似要素とli要素を横並びに
    background-color: yellow;
    padding-right: 20px;  ←疑似要素右側のスペース
  }
  • 【参照】mdn ::before (:before)
  • 【参照】olタグで括弧付きの数字で表示する
  • 参照したサイトでは、liタグに「position:relative;」、li::beforeに「position:absolute;」と「top:0;」「left:0;」を指定して調整しているが、自分は、padding-rightで調整した。
  • 括弧付き数字なら、Ready-made Counter Styleのparenthesized-decimalのほうが調整の手間がかからないが、数字が少し小さくなり、20以下しか使えない、丸括弧以外のアレンジが効かない。
